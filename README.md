# 專案名稱: Find Your Chord (找和弦)

#### 網站: https://www.yanfleet.com
#### 介紹: 網站提供使用者以兩種方式搜尋和弦，且提供非吉他愛好者創造和弦功能來打造個人專屬的旋律。
#### 核心功能: 和弦名稱搜尋、和弦按壓搜尋、創造和弦、和弦字典、會員系統
